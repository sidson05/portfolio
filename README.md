# Siddhi Songire - Developer Portfolio

A personal developer portfolio template with a sleek, cyberpunk/sci-fi theme for showcasing projects, skills, and experience. Built using React and Vite.

## 🚀 Live Demo
https://sidson05.github.io/portfolio/

## ✨ Features
- **Modern UI:** Sci-fi, modern aesthetic with glowing components and interactive animations.
- **Particle Background:** A fully immersive interactive canvas background.
- **Dynamic Projects Section:** Displays selected software, ML, and web development projects.
- **Experience Timeline:** Clean vertical timeline showcasing education and professional positions.
- **Responsive Navigation:** Smooth scroll functionality with a glass-morphic header.
- **Dark Mode Default:** Optimised colors tailored for developers.

## 🛠️ Required Technologies
- Node.js (v18+)
- Vite
- React (Preact preset)
- Vanilla CSS

## ⚙️ Quick Start

**1. Clone the repository**
```bash
git clone https://github.com/sidson05/portfolio.git
cd portfolio
```

**2. Install dependencies**
```bash
npm install
```

**3. Run the development server**
```bash
npm run dev
```

Open `http://localhost:5173/` in your browser to view the portfolio.

## 📁 Repository Structure
```
portfolio/
```
